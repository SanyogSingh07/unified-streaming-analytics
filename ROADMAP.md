## Upcoming Milestones

### v1.1.0 — Enhanced Analytics
**Target**: Q3 2025

- [ ] Country-level drill-down heatmap with choropleth rendering
- [ ] Precision-Recall and ROC curve visualizations in the dashboard
- [ ] SHAP explainability panel for feature importance
- [ ] Confusion matrix visualization component
- [ ] Correlation matrix heatmap for feature analysis

---

### v1.2.0 — Recommendation Engine v2
**Target**: Q4 2025

- [ ] Collaborative filtering (SVD-based matrix factorization)
- [ ] Hybrid recommendation system (content + collaborative)
- [ ] User preference profiles with session persistence
- [ ] A/B testing framework for recommendation strategies
- [ ] Real-time recommendation scoring API

---

### v1.3.0 — Authentication & Multi-Tenancy
**Target**: Q4 2025

- [ ] JWT-based authentication on FastAPI
- [ ] Role-based access control (Admin, Analyst, Viewer)
- [ ] User dashboard personalization
- [ ] API key management

---

### v2.0.0 — Enterprise Analytics Platform
**Target**: Q1 2026

- [ ] Real-time streaming analytics with Apache Kafka
- [ ] PostgreSQL migration from SQLite
- [ ] Redis caching layer for API performance
- [ ] Distributed training with Ray or Dask
- [ ] Power BI Embedded integration
- [ ] Cloud deployment (AWS / GCP / Azure)
- [ ] Kubernetes orchestration with Helm charts
- [ ] Prometheus + Grafana monitoring dashboards
- [ ] Automated ML retraining pipeline (MLflow)
- [ ] Data versioning with DVC
